# 🚗 Neural Network Sensor Car – Part One

This is **Part One** of a multi-phase project focused on building an AI-powered sensor car. In this phase, the foundation is laid for a self-driving vehicle using neural networks and basic sensors to perceive and react to its environment.

## 🔍 Overview
The goal of Part One is to create a working prototype of a car that can:
- Detect obstacles using sensors
- Make decisions using a neural network
- Move autonomously in a controlled space

## 🧠 Technologies Used
- Neural Network implemented in **JavaScript** (for decision-making)
- Sensor input processing and motor control logic
- Obstacle detection using Sensor

## 📌 What's Included in Part One
- Initial circuit design
- Neural network structure and input/output mapping
- Basic motor control logic
- Test simulations and live movement

## 🎯 Future Plans
- Advanced training models
- Object classification
- GPS or map-based navigation (Part Two & beyond)
